# zunaid.dev

A modern, cyber-inspired personal portfolio built for Zunaid Hasan. This static website showcases professional experience, technical skills, featured projects, publications, certifications, and contact details in a polished developer portfolio layout.

## Overview

The site is designed as a single-page portfolio with smooth scrolling navigation, animated terminal-style hero content, and a neon cyberpunk-inspired UI. It is optimized for both desktop and mobile viewing with a responsive navigation menu and interactive page sections.

## Key Sections

- **Hero**: Animated terminal introduction, social links, quick statistics, and a personal pitch.
- **Skills**: Categorized technical skills cards for frontend, backend, databases, AI/ML, QA/testing, and DevOps/communication.
- **Projects**: Highlighted projects with descriptions, tech stacks, and links to live demos or repositories.
- **Experience**: Work history, volunteer contributions, education, and career highlights.
- **Articles & Notes**: Tag-based filtering for blog posts, technical notes, and community articles.
- **About & Contact**: Certifications, contact form, and quick access to email/phone.

## Features

- Responsive layout with mobile menu support
- Animated entries and section fade-ins
- Custom terminal window design with typing animation
- Dynamic article filtering by tags
- Contact form UI with success feedback toast
- Background matrix rain effect for visual depth
- Clean, modern dark theme with neon accent colors

## Technologies

- HTML5
- Tailwind CSS via CDN
- Lucide icons for lightweight SVG icons
- Vanilla JavaScript for interactivity and animations

## Usage

1. Open `index.html` in a browser.
2. Navigate between sections using the fixed top navigation or mobile menu.
3. Use the tag filters in the Articles section to surface content by topic.
4. Submit the contact form for a sample message interaction.
